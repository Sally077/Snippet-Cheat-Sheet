Use these three properties to create a flex row layout.

Flexbox Row

.row{
display: flex;
flex-direction: row;
flex-wrap: wrap;
}

This is to create a flex box column layout.

Flexbox Column

.column{
display: flex;
flex-direction: column;
}

Build a 12-column layout using css grid.

CSS Grid Layout

.grid{
display: grid;
width: 100%;
grid-template-columns:
repeat (12 1fr);
}

This will create a background linear gradient from top to bottom.

repeat (12 1fr);
}
